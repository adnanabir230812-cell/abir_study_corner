4. Strategic Suggestions for Exam Preparation

A. Guaranteed High-Yield Topics (Prepare These First)

✓  Ecological Pyramids — draw all 3 types with labeled diagrams (Number, Biomass, Energy).

✓  Food Chain vs Food Web — comparison table + both diagrams.

✓  Energy Flow &amp; Laws of Thermodynamics — 1st and 2nd law with examples from ecosystems.

✓  Intercropping: Types + Examples — Mixed, Row, Strip, Relay with Bangladeshi examples.

✓  Intercropping vs Mixed Cropping — comparison table (objective, competition, duration, sowing, care).

✓  LER &amp; RYT — know the formula, calculate and interpret (if LER &gt; 1, intercropping is advantageous).

✓  Eutrophication — definition, mechanism, and effects on aquatic life and human health.

✓  Sundarbans — ecological significance, livelihood, biodiversity, and conservation strategies.

✓  Halophyte adaptations — vivipary, root modifications (pneumatophores, stilt/prop roots, buttress roots).

✓  Agrochemicals — threats to human health, remedial measures, water pollution linkage.

B. New/Emerging Topics (Recently Added)

★  Bioremediation (2025 — first appearance): Role in controlling environmental pollution.

★  Ratoon Cropping (2023): Difference from relay cropping.

★  Coastal vegetation distribution (2025): Know Bangladesh&apos;s coastline and species.

★  Ecological significance of Sundarbans (2025): Broader environmental angle, not just livelihood.

C. Answer Strategy Tips

→  For &apos;Assess/Evaluate/Analyze&apos; Qs: Give both sides — advantages and limitations — before concluding.

→  For &apos;Justify/Argue&apos; Qs: Start with a clear thesis, then give 3–4 logical points with examples.

→  For diagram Qs: Always label properly. Unlabeled diagrams get minimal marks.

→  For numerical Qs (LER/RYT): Write the formula first, substitute values, then interpret clearly.

→  For &apos;Compare &amp; Contrast&apos; Qs: A table format is most efficient and scores well.

→  Marks guide: [2] = definition + 1 point; [3] = 4–5 points; [4–5] = full elaboration with examples.

→  Marks guide: [3.5] (new in 2025) — treat as [4], aim for comprehensive answers.
